# CharacterArchive

## Project's Purpose
First, a little background.  I've been playing DnD for almost a year.  It's been a lot of fun.  Unfortunately, I don't really have a lot of time to dedicate to building amazing character.  I love stories, and I'm a softy for character-led games.  I thought this would be a great time to combine two things I love into a reasonably "simple" project.  Oh, how project creep can quickly balloon such projects!  Still, I'm trying to keep these initial steps small. 

1. Build a React application that clearly shows a DnD's character sheet in a presentation that fits the "moment."  For example, what do I most use when we are in the middle of conflict?  What would I like to look at when I'm trying to solve a mystery?  How should I best recall what we did last week?
2. I want this project to show my work to others who might be interested in hiring me.  As such, I will be building this project in phases.  Each phase will be focused on specific tech.  (1) <u>React</u> for the front-end, (2) <u>PostgresSQL</u> for data management, (3) <u>Java</u> for CLI backend data input, (4) <u>cloud</u> deployment application, (5) <u>GoLang</u> refactor CLI backend application, (6) <u>Fyne</u> for backend GUI application, (7) <u>React</u> for backend Web application, (8) <u>Iterate</u>
3. Provide the character building community a place where they can organize, share, and gather character-driven story ideas for TTRPGs or other such fictional writing media outputs.
